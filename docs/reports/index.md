# Reports & Analytics

CoralLedger Comply provides comprehensive reporting tools to help you analyze your VAT position, cash flow, and business performance.

## Available Reports

### Cash Flow Report
Track payment obligations, upcoming deadlines, and historical payments. See outstanding VAT amounts, upcoming due dates, and payment history.

[View cash flow reports](/docs/reports/cash-flow)

### Variance Analysis
Compare C7 customs import data against sales revenue to identify discrepancies, calculate gross margins, and flag anomalies for investigation.

[View variance analysis](/docs/reports/variance-analysis)

### Custom Report Builder
Build custom reports with configurable columns, filters, date ranges, and export formats. Save report configurations for recurring use.

[Build custom reports](/docs/reports/custom-reports)

### Shared Reports
Generate shareable report links for clients, auditors, or team members — with optional password protection and download limits.

[Share reports](/docs/reports/shared-reports)

### VAT Returns Reports
Generate, preview, and export VAT returns. See [VAT Returns](/docs/vat-returns) for details.

### Audit Reports
Review the immutable audit trail. See [Audit Reports](/docs/audit/audit-reports) for details.

## Export Formats

| Format | Use Case |
|--------|----------|
| **PDF** | Official filing, client presentations, archival |
| **CSV** | Data analysis, spreadsheet import |
| **Excel** | Formatted spreadsheets with formulas |
| **XML** | DIR submission format, system integration |
| **JSON** | Custom integrations, API consumption |

## Accessing Reports

Reports are available from the **Reports** section in the main navigation:
- Reports > Cash Flow
- Reports > Variance Analysis
- Reports > Custom Report Builder
- Reports > Audit Logs
