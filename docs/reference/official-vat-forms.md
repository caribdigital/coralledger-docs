# Official DIR VAT Forms

The Bahamian Department of Inland Revenue (DIR) publishes the official VAT forms on its own
site, [inlandrevenue.finance.gov.bs](https://inlandrevenue.finance.gov.bs/). This page is a
directory of those forms, linked **at the official source** - the DIR's site is the source of
truth for every form, and forms should always be downloaded from it directly.

CoralLedger Comply does not submit these forms for you. Where a Comply workflow produces
figures that belong on a form (for example, the return figures that map to the V.A.T. Return
worksheet), you prepare and lodge with the DIR yourself - see
[Self-Filing](/docs/getting-started/self-filing).

:::info Directory snapshot
Links below were verified reachable on 2026-07-17. If a link stops working, the DIR may have
moved or revised the form - check the DIR site directly.
:::
