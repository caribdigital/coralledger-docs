# Dashboard Tour

Learn your way around the CoralLedger Comply dashboard.

## Dashboard Overview

The dashboard provides a real-time view of your VAT compliance status:

### VAT Summary Card
- Current period totals
- Input vs Output VAT
- Net VAT payable/refundable

### Compliance Score
- Your A+ to F grade
- Trend indicator
- Areas for improvement

### Quick Actions
- Import transactions
- Generate VAT return
- View compliance alerts

## Navigation

### Sidebar Menu
- **Dashboard** - Overview and metrics
- **Transactions** - View and manage all transactions
- **Import** - Upload transaction data
- **VAT Returns** - Generate and view returns
- **Compliance** - Anomaly detection and scoring
- **Settings** - Account and business configuration

### Top Bar
- Business switcher (for multi-client users)
- Notifications
- Account menu
