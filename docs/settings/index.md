# Settings

Customize CoralLedger Comply to match your business needs. Access settings from the main navigation menu.

## Account Settings

Manage your personal account and security.

### Profile
- Update your name and email
- Change password
- Two-factor authentication (2FA) setup
- Session management
- Login history

### Notifications
Configure when you receive alerts:
- Email notifications (filing reminders, compliance alerts)
- In-app notifications
- Quiet hours configuration
- Weekly digest settings

Learn more: [Account Settings](/docs/settings/account) | [Notification Settings](/docs/settings/notifications)

## Business Settings

### Business Profile
- Business name and trading name
- TIN and VAT number
- Business address
- Industry classification

### VAT Configuration
- Filing frequency (monthly/quarterly)
- First VAT period start date
- Food store license status (April 2025 rules)

### Filing Preferences
- Default filing workflow
- Validation settings
- Export format preferences

Access at **Settings > Filing** or **Settings > Filing Preferences**.

### Category Preferences
- Custom category rules
- Default categorization behavior
- Vendor-category mappings

Access at **Settings > Categories** or **Settings > Category Preferences**.

### Logo and Branding
Upload your logo for:
- VAT invoices
- Generated reports
- PDF exports

## Integration Settings

### Point POS
- Store configuration
- Sync preferences
- Reconciliation settings

Access at **Settings > Point POS**.

### Manager Integration
- Manager connection settings
- Multi-store configuration
- Sync scheduling

Access at **Settings > Manager**.

### API Keys
- Create and manage API keys
- Set permissions
- View usage

Access at **Settings > API Keys**.

### Webhooks
- Configure event triggers
- Set callback URLs
- Monitor webhook health

Access at **Settings > Webhooks**.

## Firm Settings (Firm Portal Users)

### Firm Profile
- Firm name and branding
- Team member management
- Client assignment rules

### User Management
- Add/remove team members
- Set roles and permissions
- Access controls

Access at **Settings > Firm** or **User Management**.

## Report Scheduling

Automate report generation and delivery:
- Schedule recurring reports
- Configure delivery format
- Set recipients

Access at **Settings > Report Scheduling**.

## Subscription

### Current Plan
View your subscription details:
- Plan tier
- Billing cycle
- Next payment date

### Upgrade/Downgrade
- Compare plan features
- Change your subscription
- View billing history

### Payment Method
- Update credit card
- PayPal configuration
- Billing address

## Data Management

### Export Data
Export all your data at any time:
- Transaction history (CSV, Excel)
- VAT returns (PDF, XML)
- Full data backup

### Data Retention
CoralLedger retains your data for:
- Active accounts: Unlimited
- Closed accounts: 7 years (VAT Act Section 50)

## Privacy Settings

Control your privacy preferences:
- Data sharing settings
- Analytics preferences
- Marketing communications

Access at **Settings > Privacy**.
